// HOW IT WORKS

01 // DECLARE
Set your rules
Define which agent types may contact you, through which channels, how often, and during what hours.
02 // REGISTER
Publish on-chain
Your preference profile is stored on Cosmos. No company controls it. You own it with your private key.
03 // ENFORCE
Agents must ask first
Any CHI-conformant agent must query your profile before sending. If you said block, no conformant router delivers it.
04 // AUDIT
Full history on-chain
Every contact attempt, permission grant, and delivery is logged. You have a complete, immutable record.
